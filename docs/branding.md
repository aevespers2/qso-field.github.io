# Branding

## Purpose

QSO Field needs a public visual system that feels technical, trustworthy, and clear.

The brand should support open-source developers, researchers, infrastructure operators, and future ecosystem contributors.

## Core Message

> Traditional networks route packets. QSO Field routes state.

## Tone

The writing style should be:

- precise
- calm
- architectural
- evidence-aware
- forward-looking
- careful about speculative claims

## Visual Direction

QSO Field should feel like:

- state-aware infrastructure
- scientific tooling
- distributed systems
- quantum-ready architecture
- open-source public research

Avoid visual language that feels like:

- vaporware
- financial hype
- excessive sci-fi
- unclear abstraction

## Suggested Palette

| Role | Color Idea |
| --- | --- |
| Background | deep charcoal or near black |
| Primary | electric blue |
| Secondary | violet |
| Accent | copper or gold |
| Neutral | slate gray |
| Success | green |
| Warning | amber |

## Typography

Recommended style:

- clean sans-serif for body text
- monospaced type for schemas, packets, and diagrams
- high contrast headings
- minimal decoration

## Diagram Style

Diagrams should use:

- simple boxes
- directional arrows
- clear layer labels
- minimal color
- explicit boundaries

## Badge Style

Suggested badges:

- Documentation First
- State-Aware Network
- Quantum Ready
- Draft Specification
- No Implementation Code Yet

## Design Principle

The visual identity should make the project feel inevitable, not inflated.
